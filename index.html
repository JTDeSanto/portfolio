Temperature Conversion

This ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 

puts "Celcius to Farenheit Conversion"
puts "Please Enter Temperature in Celcius:"
celcius = gets.chomp

def celcius_to_fahrenheit (temp)
  return (temp.to_i * 1.8) + 32
end

fahrenheit  = celcius_to_fahrenheit(celcius)

puts "You have #{fahrenheit} degrees fahrenheit"

def ordinal (n)
  number = n.to_i
  
  if number % 100 == 11 || number % 100 == 12 || number % 20 == 13
    return "#{number}th"
  elsif  number % 10 == 1
    return "#{number}st"
  elsif  number % 10 == 2
    return "#{number}nd"
  elsif number % 10 == 3
    return "#{number}rd"
  else 
    return "#{number}th"
  end
  
end

puts "The number #{input}'s ordinal is #{ordinal(input)}"

Foobar

This ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 


puts "Enter a number:"
input = gets.chomp.to_i
fooBar = []
n = 0

while n != input
  n += 1
    
  if n % 3 == 0 && n % 5 == 0
    fooBar<<("FooBar")
  elsif n % 3 == 0
    fooBar<<("Foo")
  elsif n % 5 == 0
    fooBar<<("Bar")
  else
    fooBar<<(n)
  end

end

puts fooBar
